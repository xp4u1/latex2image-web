Convert LaTeX math equations to PNG/JPG/SVG images

LaTeX2Image allows LaTeX math equations to be exported directly to multiple image formats, and saved for use in other documents.

Equations are very time consuming to enter in word. the math mode of LaTeX is the master in this, but unfortunately Word does not support LaTeX very much. in newer versions, but also only under windows, you can switch to the LaTeX input mode. nevertheless you can only use a fraction of the commands provided by LaTeX. therefore i programmed/improved this converter, so that you only need to drag the image of the formula into the word document. The complete source code is available on GitHub.
